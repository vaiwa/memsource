# Memsource Assignment

## init
``` bash
npm install
npm run compile

# and create `.env` file with your own *userName* and *password*:
echo 'LOGIN=userName' > .env
echo 'PASSWD=password' >> .env
```

## start
```bash
npm start
```
and visit [localhost:8080](http://localhost:8080)

## run integration tests
```bash
npm test
```
